# Changelog

All notable changes to ClaudeBar will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

## [0.2.4] - 2025-12-25

### Fixed
- Rebuild appcast.xml without old entries
- Remove build number check from update-appcast

## [0.2.3] - 2025-12-24

### Fixed
- Use GitHub run number as build number for consistent versioning

## [0.2.2] - 2025-12-23

### Added
- Sparkle auto-update framework integration
- Modern landing page design

### Fixed
- Various stability improvements

## [0.2.1] - 2025-12-22

### Added
- Initial Sparkle setup for auto-updates

## [0.2.0] - 2025-12-20

### Added
- GitHub Copilot usage monitoring
- Gemini API quota tracking
- System notifications for quota status changes

### Changed
- Improved menu bar UI design

## [0.1.0] - 2025-12-15

### Added
- Initial release
- Claude CLI usage monitoring
- Codex CLI usage monitoring
- Menu bar interface with quota display
- Automatic refresh every 5 minutes
